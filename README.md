# Calendar

Calendar for the current year

--------------------------------

### Run

Download repository:

    $ git clone https://github.com/YuriShvetsov/calendar-js folder

Install dependencies:

    $ npm i

Server with hot reload at http://localhost:9000/:

    $ npm run dev

Output will be at dist:

    $ npm run build
